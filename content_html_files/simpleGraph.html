Simple Graph

A simple graph is a graph that does not contain more than one edge between the pair of vertices. If each pair of nodes or vertices in a graph G=(V, E) has only one edge, it is a simple graph. As a result, there is just one edge linking two vertices, depicting one-to-one interactions between two elements.

There are four vertices here labeled B, C, E, F. Each vertex appears to be connected to others with straight edges. It is a simple graph because it does not contain any self-loops (edges connecting a vertex to itself) or multiple edges between any two vertices.

Applications

  • Social Networks: Represent friendships, where each edge signifies a connection without self-loops or multiple edges.
  • Transportation Networks: Cities or stations as nodes connected by edges that represent direct paths without redundant links.
